## About

This simple Python script uses the [fpdf](https://pyfpdf.github.io/fpdf2/) library to create a **well-formatted PDF** of tough English vocabulary words and their meanings.  
Instead of manually typing hundreds of vocabulary words by hand, you can now generate a clean, printable PDF in seconds!

Perfect for college students, teachers, and anyone learning advanced English words.

## How to Use

1. Make sure you have Python installed (version 3.x recommended)  
2. Install the required library by running:  
```

pip install fpdf

```
3. Edit the Python script (`vocab_generator.py`) to add or update your vocabulary words  
4. Run the script:  
```

python vocab\_generator.py

```
5. Find your generated PDF named `tough_vocab.pdf` in the same folder!

---

## Customize

- Add your own words in the dictionary inside the script or load them from a JSON file  
- Adjust font size, style, and layout easily in the script (using `set_font` and `multi_cell`)  
- Increase the number of words or randomize selection (if using a larger word bank)

---

## Example Output

The PDF will have:

- A title at the top, centered and bold  
- Each vocabulary word followed by its definition  
- Text wrapping automatically for long definitions
